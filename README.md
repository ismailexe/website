# ismailexe — Personal Portfolio

Personal portfolio website built with **React + Vite** on the frontend and **Express** on the backend, featuring an AI-powered chat assistant.

## Tech Stack

- **Frontend:** React 19, Vite, CSS
- **Backend:** Node.js, Express
- **AI:** Groq API (llama-3.3-70b-versatile)

## Features

- About, Skills, Projects, Education, and Contact sections
- AI chat widget powered by Groq (streaming responses)
- Hidden easter egg for Çankaya University students

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Set up environment

Create a `.env` file in the root directory:

```env
GROQ_API_KEY=your_groq_api_key_here
```

### 3. Run the project

```bash
npm run dev
```

This starts both the Vite dev server (`localhost:5173`) and the Express backend (`localhost:3001`) concurrently.
